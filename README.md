# RoofIQ Clean Start

Static Vercel-ready dashboard. No npm, no Next.js, no build step.

## Deploy

### Option A: Vercel drag-and-drop
1. Unzip this folder.
2. Go to Vercel.
3. Add New Project -> Deploy manually.
4. Drag the unzipped folder contents.

### Option B: GitHub
1. Create/clean a repo.
2. Upload these files to the repo root.
3. Import the repo into Vercel.
4. Deploy.

## Edit data
Update `market-data.js`.

## Files
- `index.html`: dashboard layout
- `styles.css`: styling
- `app.js`: filtering, map, charts
- `market-data.js`: market dataset
- `vercel.json`: static routing
